## ピカチュウにする
まずHyperの公式サイトからダウンロードします。
そしてhyper-pokemonをインストールします。
最後にpokemonの設定をしたら完了です。  

```zsh
$ hyper i hyper-pokemon
```

~/.hyper.js
```javascript
config: {
    // pokemon
    pokemon: 'pikachu',
    unibody: 'true',
}
```
